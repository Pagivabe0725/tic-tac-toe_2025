# Tic Tac Toe – Frontend

Frontend application for a Tic Tac Toe game, responsible for UI, state management, and client-side game interaction.

Built with modern Angular features (Signals, zoneless change detection) and scalable state management using NgRx.

## 🚀 Live Demo

https://tic-tac-toe-2025-mu.vercel.app/tic-tac-toe

## 🧰 Tech Stack

- Angular (Signals, zoneless change detection)
- NgRx (state management)
- SCSS

## ✨ Features

- Dynamic board size (3×3 up to 9×9)
- Local two-player mode or play against an AI opponent
- 4 AI difficulty levels (very easy → hard)
- Save and load games (available for authenticated users)
- User authentication (required for AI gameplay and saved matches)
- Predictable state management using NgRx + Signals

## 🧠 How It Works

The application handles user interaction, rendering of the game board, and communication with the backend API.

Game state is managed using NgRx and Angular Signals, ensuring predictable updates and efficient UI rendering.

Authenticated users can access advanced features such as AI gameplay and saved matches.
